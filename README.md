# Maria Pop's Portfolio
Welcome to my personal portfolio! This website showcases my projects, skills and experiences as a web developer. Explore the site and feel free to get in touch!

## Technologies Used

- React.js
- CSS
- HTML

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/<your-github-username>/portfolio
```

2. Navigate to the project directory:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
